# 任务4：IP电话系统

## 项目概述

本任务是计算机网络专题实践的第四个任务，在任务3的点对点实时语音聊天工具基础上，实现了：
1. 用户注册/登录系统（用户名唯一标识，密码与用户名相同）
2. 联系人增删改查功能
3. 数据持久化存储到JSON文件
4. 具备"拨号-接听-挂断"完整流程的IP电话系统
5. GUI显示空闲 → 呼叫中 → 通话中 → 结束的状态流转

## 文件说明

- `telephone_protocol.py` - 应用层协议定义
- `data_store.py` - 数据存储模块（用户账号和联系人持久化）
- `telephone_server.py` - 服务器实现
- `telephone_client.py` - 客户端核心
- `client_gui.py` - 客户端图形界面
- `server_gui.py` - 服务器图形界面
- `tk_utils.py` - Tkinter工具函数
- `client_gui_start.py` - 客户端启动器
- `server_gui_start.py` - 服务器GUI启动器
- `server_start.py` - 服务器命令行启动器

## 运行说明

### 启动服务器

```bash
# 图形界面版
python server_gui_start.py

# 命令行版
python server_start.py
```

服务器默认监听端口：8881

### 启动客户端

```bash
python client_gui_start.py
```

## 功能使用

### 1. 用户注册/登录

- 首次使用时，在登录界面输入用户名和密码
- 点击"注册"按钮创建新账号（密码自动与用户名相同）
- 点击"登录"按钮登录已有账号

### 2. 联系人管理

登录后进入"电话本"标签页：
- 点击"添加联系人"添加新联系人
- 点击"删除联系人"删除选中联系人
- 点击"修改联系人"修改选中联系人
- 在搜索框输入关键词，点击"查找"搜索联系人

### 3. IP电话

进入"IP电话"标签页：
- 在输入框输入对方用户名，点击"呼叫"发起通话
- 收到呼叫时，点击"接听"或"拒绝"
- 通话中可随时点击"挂断"结束

### 状态流转

- 空闲（灰色）- 可以发起呼叫
- 呼叫中（橙色）- 等待对方接听
- 响铃中（蓝色）- 收到对方呼叫
- 通话中（绿色）- 正在进行语音通话
- 通话结束 - 通话已结束

## 数据存储

用户数据和联系人数据保存在 `data` 目录下：
- `data/users.json` - 用户账号信息
- `data/contacts/` - 各用户的联系人列表
